# Uso de [!UICONTROL Recency] e [!UICONTROL Frequency] em segmentos

Use [!UICONTROL Recency] e [!UICONTROL Frequency] para fornecer seus parâmetros de segmento de quantas vezes um visitante precisa se qualificar para uma característica em um determinado período de tempo. Excelente para afinidade de conteúdo e casos de uso de limite de frequência, bem como outros.
